Stop-UAT Criteria

Pause UAT and escalate immediately if any of the following are encountered:

  • Data integrity bugs (totals do not add up)
  • Auth/role bypass (cashier accessing admin features)
  • Payment or order total mismatches
  • Receipt or report calculation errors
  • Blank pages or unhandled JavaScript errors in console
